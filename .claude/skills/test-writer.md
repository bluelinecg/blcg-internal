name: test-writer
description: Write comprehensive tests covering behavior and edge cases.
---

Use this skill when writing or improving tests.

---

## Scope

- Unit tests
- Component tests
- End-to-end tests

---

## Principles

- Test behavior, not implementation
- Cover edge cases
- Mock external services

---

## Requirements

- Co-locate tests with source files
- Use proper test naming
- Avoid brittle tests

---

## Checklist

- Happy path covered
- Edge cases covered
- Error cases covered

---

## Goal

Ensure reliability and prevent regressions.
